E) PHP Files - top

In Bonfire Template there is only one PHP file included. That file is mail.php file which is used to send message from contact page's contact form to the given address.

How to customize contact form setting.

Contact form is fully functional regarding sending submitted information to given address. To change e-mail address for receiving e-mail and customize that message, you have to open mail.php file given in main directory. In this file you can see '$address = "email@server.com"', you just have to change this e-mail address to your required e-mail and you will receive emails from this contact form on that address. You can change body text for that message in next lines. When you are done, save and close file.
